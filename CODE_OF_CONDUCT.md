# 社区行为准则 / Code of Conduct

USB LinkMic 欢迎不同经验、背景和观点的参与者。参与 Issue、Pull Request、讨论或其他项目空间时，请遵守以下原则：

- 围绕技术事实和用户影响讨论，对事不对人。
- 尊重他人的时间、隐私、设备限制和语言差异。
- 不接受骚扰、歧视、威胁、侮辱、人肉搜索或发布他人私密信息。
- 给出可执行的反馈；在意见不一致时允许维护者基于项目范围、安全性和维护成本做最终决定。

不适合公开报告的行为问题，请通过仓库所有者 [GitHub 主页](https://github.com/xxkingstuggle)列出的私下联系方式反馈，不要在公开 Issue 中披露敏感信息。如果主页没有可用的私下联系方式，可以创建一个不含敏感细节的 Issue，请维护者提供私下沟通渠道。维护者可以编辑或删除不当内容，并限制持续违反本准则的参与者。

---

Be respectful, discuss technical facts and user impact, protect other people's privacy, and do not engage in harassment, discrimination, threats, or publication of private information. Sensitive conduct reports should be sent privately using the contact method listed on the repository owner's GitHub profile.
