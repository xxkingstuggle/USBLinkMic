# Security Policy

## Supported Versions

Security fixes are provided for the latest public release.

## Reporting a Vulnerability

Please do not disclose security issues publicly before they are fixed. Open a private GitHub security advisory if available, or contact the maintainer through GitHub.

Include:

- affected version or commit
- platform and OS version
- reproduction steps
- expected impact

---

# 安全策略

## 支持版本

安全修复面向最新公开版本提供。

## 报告漏洞

在修复前请不要公开披露安全问题。请优先使用 GitHub private security advisory，或通过 GitHub 联系维护者。

请尽量包含：

- 受影响的版本或提交
- 平台和系统版本
- 复现步骤
- 可能影响
